# 获取帮助 | Support

## 使用问题或功能建议

请创建 GitHub Issue，并使用合成数据描述：

- 你想解决的经营问题；
- 当前输入格式；
- 预期结果；
- 实际结果；
- 运行命令和报错信息。

## 方向讨论

不确定是否属于 Bug 或功能时，使用 GitHub Discussions。

## 安全或隐私问题

不要创建公开 Issue。请按 [SECURITY.md](SECURITY.md) 处理。

## 不提供的支持

本开源项目不承诺：

- 帮助绕过平台登录、验证码或风控；
- 接收真实商户账号和数据进行公开调试；
- 对第三方平台的服务条款作法律判断；
- 为经营、财务或税务结论承担最终责任。

---

For usage questions and feature requests, open an Issue with synthetic data and a reproducible example. Use Discussions for broader ideas. Never post credentials, merchant data, or private security details publicly.
